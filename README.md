# TreePy

## How To Install

### MacOS/Linux
> Type: `pip3 install treePy` into a terminal and press enter.<br>
> `import treePy as tp`
### Windows
> Type: `pip install treePy` into CMD and press enter.<br>
> `import treePy as tp`

## What is TreePy
TreePy is an open source python which simplifies linked trees in python
